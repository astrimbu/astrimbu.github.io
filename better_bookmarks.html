***EDIT: BASICALLY BOOKMARK DATABASE EXPLORER 2000

Partial replacement for built-in bookmark functionality.

Not a full replacement because you still want the bookmarks bar.

This app is for all the webpages you visit and want to remember at a later time.

Instead of losing pages in the Bookmarks folder hierarchy, this app creates and
maintains a customizable database for you to query.

On icon click:
    Tag:
    Description:
    Color: x x x x x x x x x
    Group: o o o o o o o o o
    Done [enter]

You only have to enter as much information as you want before hitting Done.

    Filter bar: limit tuples matched by attributes
